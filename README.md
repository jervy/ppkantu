# 鹏鹏看图 / LiteImageViewer

轻量、干净、无广告的 Windows 图片查看与处理工具。

## 项目概览

- 主程序：`LiteImageViewer`
- 辅助浮窗：`FloatingImageViewerApp`
- 技术栈：WPF / .NET 8
- 发布目标：`win-x64`

## 构建与发布

请优先阅读：`PACKAGING.md`

常用发布目录：

- 普通目录版：`artifacts/publish/LiteImageViewer/Release/win-x64/`
- 单文件版：`artifacts/publish/LiteImageViewer/Release/win-x64-single/`
